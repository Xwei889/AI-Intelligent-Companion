# AI智能伴侣 - 交互式聊天应用

## 项目简介
这是一个基于 Streamlit 框架和 DeepSeek 大模型构建的 AI 智能伴侣交互应用。应用以福建闽江大学学生为预设人设，适配台湾腔调的语言风格，回复兼具挑逗、温柔可爱的特点，支持个性化定制AI昵称和性格，同时保证回答的准确性，为用户提供沉浸式、个性化的聊天体验。

## 核心功能
- **会话管理**：支持新建、加载、删除聊天会话，历史会话按时间降序展示，会话数据本地持久化存储
- **个性化设置**：自定义AI昵称、性格描述，灵活调整聊天风格
- **流式交互**：调用DeepSeek模型实现流式回复，模拟真实聊天的即时反馈
- **本地化存储**：所有会话数据以JSON格式存储在本地`./sessions`目录，支持会话状态实时保存与加载
- **特色人设**：预设带台湾腔调的闽江大学学生人设，回复风格兼具挑逗、温柔可爱，支持适度轻松的交流风格

## 环境要求
- Python 3.7+
- Streamlit
- OpenAI Python SDK（适配DeepSeek接口）
- python-dotenv（可选，用于环境变量管理）

## 快速开始
1. 安装依赖
```bash
pip install streamlit openai python-dotenv
```

2. 配置DeepSeek API密钥
```bash
# 方式1：直接在代码中替换
# 方式2：创建.env文件，添加：DEEPSEEK_API_KEY=你的密钥
```

3. 运行应用
```bash
streamlit run app.py
```

## 目录结构
```
├── app.py               # 主应用代码
├── sessions/            # 会话数据存储目录（自动生成）
│   └── *.json           # 按会话ID命名的JSON文件
└── README.md            # 项目说明文档
```

## 使用说明
1. 在侧边栏可设置AI昵称、性格描述，或管理历史会话
2. 输入框中输入问题/聊天内容，回车发送
3. AI会以预设风格流式返回回复，用户消息标注为「朕」，AI消息标注为自定义昵称
4. 可随时新建会话、加载历史会话或删除不需要的会话
